1) 🖥️ Process State Simulator
A C program that simulates how an operating system manages process state transitions in real time — without freezing the program while displaying output, just like a real OS!
---
📌 About the Project
1) In an operating system, every process goes through a series of states during its lifetime. This program demonstrates those state transitions interactively. You can press any key at any time to check the **current state** of the running process — the process keeps moving in the background, just like real OS scheduling.
---
🎮 Usage
Run the program
1. The process starts automatically from the `NEW` state
2. Press any key at any time to print the current state
3. Watch the process complete all states and terminate
---
   💡 Key Concepts Demonstrated
1. Process lifecycle in operating systems
2. Non-blocking input using `kbhit()`
3. State machine implementation using `enum` in C
4. Real-time simulation without halting program execution
